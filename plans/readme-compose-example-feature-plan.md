# README Compose Example Feature Plan

## Goal

Add a production/container usage example to the README that reflects the supported deployment model.

## What is required

- A concise compose example in the README
- Guidance on the required env vars
- Links to the full production compose file and deployment docs

## Plan

1. Add a short production compose example to the README.
2. Describe how to configure image tags, OIDC env vars, and persistence.
3. Link the example to the full production compose file and deployment guide.

## Acceptance criteria

- The README contains a production compose example.
- The example matches the real production compose structure.
- The docs link readers to the full deployment instructions.
- The feature includes examples that are discoverable from the project root.
